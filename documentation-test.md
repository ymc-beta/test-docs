# 1.3 INSTALLATION

## 1.3.1 SCOPE OF DELIVERY
* E-Control software
* Activation code for installation on a computer
* - or -
* Link to the licensing server

## 1.3.2 REQUIREMENTS
* Microsoft Windows 10 or newer
* Min. 8 GB RAM
* Administrator rights on the respective computer
* Up-to-date web browser for online licensing
* Optional
    * Touchscreen
    * Barcode / RFID
    * Printer

## 1.3.3 PERFORMING INSTALLATION

1. Unzip the software zip file.
2. Run "Lista-eControl-1.0-Setup.exe" and follow the instructions of the installation wizard.
   The software is then installed.
   If the "Launch LISTA E-Control" option is activated in the "Install" window, E-Control will start automatically after installation and open the window for licensing where you enter the activation code.

### Licensing by entering the activation code

1. Click **Online Licence Activation**.
2. Enter the activation code and click **ACTIVATE**.

### Licensing via link to the licence server

1. Open the browser and enter the link provided for licensing.
   The licence server page is then displayed.
2. Select the licensing type and follow the instructions of the licence server.

### Licensing to a computer

1. Select licence.
2. Select CmContainer: **Retrieve CmContainer automatically**
3. Select **Update selected licences now** to complete licensing.
   After licensing has been completed, the message "Licence transfer successfully completed" will appear.
4. Complete licensing by pressing **OK**.
   The **Database state** window then appears.
