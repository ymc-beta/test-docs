## 5.6 Importing Cabinets via MQTT

For electronic cabinets connected via MQTT, the cabinet structure can be automatically imported from the hardware.

### MQTT Cabinet Import

1. Navigate to **Settings > Cabinets > Import MQTT**.
2. The system discovers connected cabinets via the MQTT broker.
3. A visualization of the discovered cabinet structure is shown.
4. Review the discovered cabinets and drawers.
5. Confirm the import.

The cabinet and drawer structure is automatically created in the system based on the hardware configuration

> **Note:** MQTT settings must be properly configured in **Settings > MQTT** before using this feature. See [Section 7.3](07-settings.md#73-mqtt-configuration).
