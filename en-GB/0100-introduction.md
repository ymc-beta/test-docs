# Introduction

## About LISTA E-Control

LISTA E-Control is an electronically controlled warehouse management system for content in a LISTA cabinet system and offers the following advantages:

- **Access control** down to the drawer level, with electronic lock/unlock
- **Full traceability** of all storage and removal processes through comprehensive audit logging
- **Product lending management** for tracking borrowed tools and equipment
- **Low stock monitoring** with configurable warning and alarm thresholds
- **Pick list processing** for efficient batch operations
- **Report generation** for stock, consumption, and low-stock analysis
- **Touch-screen optimized** interface designed for a wide variety of environments

## About this documentation

This documentation is a technical manual that describes the operation and administration of the LISTA E-Control system.
