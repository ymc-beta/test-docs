# Overview of the LISTA E-Control interface


> [!NOTE]
> **Touch-Screen Operation**
> 
> LISTA E-Control is optimized for touch-screen operation:
> - All buttons and interactive elements are sized for finger touch
> - A virtual on-screen keyboard is available and can be toggled from the navigation bar
> - Scrolling (horizontal and vertical) and works with standard touch gestures


## Navigation bar

The navigation bar at the top of the screen provides easy access to all main features:

### Left section (product navigation)

| Button         | Function                                      |
| -------------- | --------------------------------------------- |
| ***Scan***     | Product search via barcode or RFID scan       |
| ***Products*** | Product browsing via categories               |
| ***Storage***  | Product browsing via cabinet/storage location |


### Center section (user info)

| Element           | Function                                                                 |
| ----------------- | ------------------------------------------------------------------------ |
| Username          | Shows the name of the currently logged-in user                           |
| Language selector | Switch between the supported languages for the LISTA E-Control interface |
The language selector allows the selection of the following languages:

- English (EN)
- German (DE)
- Spanish (ES)
- French (FR)
- Italian (IT)
- Chinese (ZH)


### Right section (system controls)

| Button         | Function                                               |
| -------------- | ------------------------------------------------------ |
| ***Keyboard*** | Show/hide the on-screen virtual keyboard               |
| ***Help***     | Open the built-in help documentation                   |
| ***Settings*** | Access user settings and system administration options |
| ***Log out***  | Log out of the system                                  |
